Dit project is een visuele recreatie van YouTube, gebouwd met HTML en CSS. Het biedt een responsief ontwerp dat zich aanpast aan verschillende schermformaten, wat een gebruiksvriendelijke ervaring mogelijk maakt op zowel desktop als mobiel. Hoewel de focus ligt op de lay-out en visuele presentatie, geeft het project een goed beeld van de structuur en opmaak van een moderne videostreaming-site.
